# Portfólio - Luan Gerber Siiss

Este repositório contém o código-fonte do meu site de portfólio profissional, onde apresento meus projetos, habilidades e informações sobre minha trajetória na área de desenvolvimento de software.

## Sobre o projeto

O objetivo deste site é reunir e apresentar alguns dos projetos que desenvolvi durante meus estudos e experiências práticas na área de tecnologia. Nele é possível visualizar detalhes dos projetos, tecnologias utilizadas e acessar os respectivos repositórios.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
